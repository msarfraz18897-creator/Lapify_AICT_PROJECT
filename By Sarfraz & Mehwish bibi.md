## Lapify – Laptop & Accessories Suggester.
## Project Overview

Lapify is a console-based Python application designed to help users select suitable laptops and accessories based on their needs. It simplifies decision-making by organizing laptops into categories, displaying specifications, and recommending accessories through a user-friendly menu-driven interface.

## Problem Statement

Many users purchase laptops based on market trends instead of their actual requirements. This often leads to overspending or buying devices with insufficient performance. Lapify solves this problem by providing a structured system for making informed decisions.

## Objectives
Provide laptop recommendations based on categories
Display brand-wise laptop details
Suggest useful accessories
Implement a menu-driven console interface
Use JSON for structured data storage
Reduce incorrect purchasing decisions
## Scope
Console-based system (Text UI)
Predefined laptop categories and brands
JSON-based data storage
Basic input validation

Future enhancements may include GUI support, live pricing, and online database integration.
